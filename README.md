## :information_source: Como baixar o projeto

```bash
#Clonar o repositório
$ git clone https://github.com/CurtisYoung/node_masterclass_rs.git

# Entrar no diretório
$ cd node_masterclass_rs

# Instalar as dependências
$ npm install

# Inicializar o projeto
$ npm run api
$ npm start
```
